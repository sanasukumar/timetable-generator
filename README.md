# timetable-generator

Browser-only academic timetable generator.

## Stack
- HTML
- Vanilla JS
- TailwindCSS
- SheetJS
- JSONata

## Features
- Excel-driven scheduling
- Weekly recurring timetable generation
- Constraint-aware heuristic scheduler
- Batch/teacher/room conflict detection
- Excel export
